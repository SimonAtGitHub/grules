Package org.grules.script.expressions

This packages contains terms classes used in a rule definition.

 

Interface Summary
Term A rule term.
 

Class Summary
AndTerm A term which is a conjunction of two other terms.
BinaryValidationTerm A validation term that combined from two other terms via a boolean operation of arity two, for example disjunction or conjunction.
ClosureTerm A term implemented as a closure.
FunctionTerm A term implemented as a function call.
NotTerm A term that represents a negation operator on a validation term.
Operators Operators used in rule expressions: and, or, chaining operator (>>), binding an error ([])
Operators.ClosureOperators
Operators.SubrulesSeqOperators >> operator that combines two subrules in a sequence such that a result of application of a first subrule is passed to a second subrule.
Operators.TermOperators
Operators.TildeTermOperators
OrTerm A term which is a disjunction of two other terms.
Subrule A subrule expression (parts of a rule between ">>" operators).
SubruleFactory Factory for conversion and validation subrules.
SubrulesSeq A sequence of subrules.
SubrulesSeqWrapper
TermWrapper Wraps a closure into a term.
TildeTerm A conversion term for boolean expressions.
UnaryTerm A term that consists of an operator and a term to which it has to be applied, for example a negation operator.
 

Exception Summary
InvalidBooleanTermException Thrown to indicate that the term returns not a boolean type.
InvalidSubruleException Signals an invalid subrule expression.
InvalidSubrulesSeqException Signals an invalid subrules sequence.
InvalidTermException Signals an invalid term expression.
InvalidValidatorException Signals an invalid validator.
